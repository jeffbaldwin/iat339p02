Submit / Reset Button for Forms

Class Name

.submit_form
.reset_form

Sample Code

<form action="#" method="post">
   <button class="submit_form" type="submit" value="Submit">Submit</button>
   <button class="reset_form" type="reset" value="Reset">Submit</button>
</form>
